Instructions:
  1. Download the file Office 2013-2019 C2R Install v6.7.0.zip to a folder which is excluded from all antiviruses.
  2. Extract the contents of the archive into the folder where you downloaded the file with WinRAR or 7-zip or an archival tool of your choice. Password = 0123456789.
  3. Double click on the file OInstall.exe.
  4. You will see this window. In the top left dropdown, select the edition of Office you want to install.
  5. In the next dropdown, select ProPlus. If installing Office 2019, select ProPlus 2019 and NOT ProPlus 2019 Volume.
  6. Choose between the full Office packs or single applications by ticking either of these two checkboxes.
  7. If you select the full Office pack (left) then I recommend choosing ProPlus in the edition dropdown.
  8. If you select the single applications (right) then check the applications you want to install.
  9. Don't change the Update Channel dropdown at the lower area of the window.
  10. Choose between x86 or x64 in the top area of the window (x86 = 32-bit)
  11. Select the languages you want in the rightmost column.
  12. Click on the Install Office button.
  13. When the process has finished, click on the Utilities tab.
  14. Tick the Online KMS and Office reactivation task boxes.
  15. Click on the Office RETAIL => VL button (for all editions, including 2019 VL)
  16. When the process has finished, click the Activate Office button.
  17. When the process has finished, restart your PC.
  18. Exclude following folders and files from all antivirus programs (It will exclude from Windows Defender automatically, and if some of the items below aren't present, that's fine):
  • C:\Windows\files
  • C:\Windows\System32\Tasks\OInstall
  • C:\Windows\files\bin\kmss.exe
  • C:\windows\OInstall.exe
  • C:\Windows\AutoKMS
  • If prompted by Windows Firewall, tick all boxes and click Allow for C:\Windows\files\bin\kmss.exe. If you don't get such a prompt, that's fine.
FAQ:
Do I need to keep the downloaded OInstall.exe? / My antivirus detected OInstall.exe, what should I do?
  • If you followed the instructions correctly (i.e. ticking the Online KMS and Office reactivation task boxes, then you can safely delete the downloaded OInstall.exe file (but not the one in C:\Windows), either manually or via your antivirus if it was detected.
  • The detection is a false positive - this tool is just an Office Installer packed with the well known KMS method of activating Office. It is created by Ratiborus, well known author of many KMS tools.
  • The activation of Office, and Office itself, is completely independent of the downloaded OInstall.exe file (but not the one in C:\Windows) after installation and activation.
  • Please note that when you run the activator OInstall.exe will drop itself and it's resources to C:\Windows\System32, and this is normal. DO NOT DELETE THE FILES FROM THIS LOCATION.
